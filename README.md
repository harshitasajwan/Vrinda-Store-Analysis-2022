# Vrinda Store Sales Analysis (Excel Dashboard Project)

## 📊 Project Overview

This project analyzes **Vrinda Store's 2022 sales data** using Microsoft Excel.
The goal of the project was to clean the dataset, analyze customer purchasing behavior, and build an **interactive sales dashboard** to generate business insights.

The dashboard helps understand:

* Sales trends across months
* Customer demographics
* Order status distribution
* Sales performance by state
* Sales channels performance

---

## 📁 Dataset

The dataset contains sales transactions from **Vrinda Store (2022)** with information such as:

* Order ID
* Customer ID
* Gender
* Age & Age Group
* Order Date & Month
* Order Status
* Sales Channel
* Product Category
* Quantity
* Amount
* Shipping City & State

Total columns: **21**

---

## 🧹 Data Cleaning

Data preparation steps performed in Excel:

* Removed duplicates
* Formatted date columns
* Created **Age Groups**
* Verified missing values
* Standardized category names
* Created pivot tables for analysis

---

## 📈 Analysis Performed

The following business questions were explored:

1. **Sales vs Orders Trend**
2. **Men vs Women Purchase Contribution**
3. **Order Status Distribution**
4. **Top States by Sales**
5. **Age Group vs Gender Purchasing Behavior**
6. **Sales by Channel**

---

## 📊 Dashboard Features

The Excel dashboard includes:

* Monthly **Sales vs Orders chart**
* **Gender-wise sales comparison**
* **Order status breakdown**
* **Top performing states**
* **Age group vs gender analysis**
* **Sales channel distribution**

Interactive filters/slicers allow users to explore the data dynamically.

---

## 🔍 Key Insights

Some insights from the analysis:

* **Women customers contribute more to total sales than men**
* The **adult age group (30–49)** is the highest purchasing segment
* **Maharashtra, Karnataka, and Uttar Pradesh** are top-performing states
* Most orders are **successfully delivered**
* Major sales channels include **Amazon, Flipkart, and Myntra**
